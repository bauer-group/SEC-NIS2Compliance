# 模板：供应商评估

## 说明

根据 §30 第2款第4项 BSIG 评估服务提供商和供应商的安全成熟度。在签订合同前进行，至少每年重复一次。

## 注意事项

- **严重**供应商每**半年**重新评估
- **高**风险供应商每**年**重新评估
